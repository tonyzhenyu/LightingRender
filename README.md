### blinn-phong 传统经验光照模型

blinn-phong 光照模型

- 环境光项 用常量表示(用IBL可以替换该常量项)
- 漫反射项 (Diffuse)
    - 兰伯特余弦定律(Lambert Consine Law)中指出，漫反射的大小取决于表面法线和光线的夹角，当夹角越大时，漫反射分量越小，当夹角接近90度时，我们认为漫反射几乎为零。
- 高光反射项 (Specular)
    - 描述光线从入射到表面出射后有多少光线在视角向量的投影值
